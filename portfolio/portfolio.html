Microsoft Word Manual: Microsoft Word for Marketers: A Time-Saving Guide

Microsoft Word for Marketers: A Time-Saving Guide is a modular software documentation manual designed for marketing professionals and students. As my first long-term technical writing project, this guide breaks down essential Microsoft Word 2024 for Mac features, focusing on efficiency, organization, and document design. This project strengthened my ability to describe software functions into clear, actionable steps, demonstrating skills in technical communication, content structuring, and user-focused documentation. I also learned the process of user research by interviewing Subject Matter Experts and usability by conducting usability tests. This guide highlights my growth in problem-solving, instructional design, and creating an accessible resource that enhances productivity for a target audience.

This project highlights my skills in critical analysis, project management, and persuasive communication to advocate for global literacy initiatives.
